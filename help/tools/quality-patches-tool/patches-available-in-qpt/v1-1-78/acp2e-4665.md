# ACP2E-4665: i prodotti secondari di prodotti configurabili con video nelle gallerie di prodotti non vengono visualizzati quando richiesto tramite l’API REST

La patch ACP2E-4665 risolve il problema se i prodotti secondari di un prodotto configurabile non vengono restituiti dall’API REST quando un video viene incluso nella galleria di prodotti. Questa patch è disponibile quando è installato [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78. L’ID della patch è ACP2E-4665. Questo problema è pianificato per la risoluzione in Adobe Commerce 2.4.9.

## Prodotti e versioni interessati

**La patch è stata creata per la versione di Adobe Commerce:**

* Adobe Commerce (tutti i metodi di implementazione) 2.4.8-p3

**Compatibile con le versioni di Adobe Commerce:**

* Adobe Commerce (tutti i metodi di implementazione) 2.4.8 - 2.4.8-p4

>[!NOTE]
>
>La patch potrebbe diventare applicabile ad altre versioni con le nuove versioni di [!DNL Quality Patches Tool]. Per verificare se la patch è compatibile con la versione di Adobe Commerce in uso, aggiornare il pacchetto `magento/quality-patches` alla versione più recente e verificare la compatibilità nella pagina [[!DNL Quality Patches Tool]: Cerca patch](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html). Utilizza l’ID patch come parola chiave di ricerca per individuare la patch.

## Problema

L’API REST non restituisce i prodotti secondari di un prodotto configurabile quando un video è presente nella galleria di prodotti.

<u>Passaggi da riprodurre</u>:

1. Aggiungi un video al prodotto secondario di un prodotto configurabile.
1. Invia una richiesta al seguente endpoint API REST:
   `rest/v1/configurable-products/<PARENT_CONFIGURABLE_PRODUCT_SKU>/children`.

<u>Risultati previsti</u>:

L’API REST restituisce informazioni sul prodotto.

<u>Risultati effettivi</u>:

L’API REST restituisce il seguente errore:

```
Error: Call to a member function getVideoProvider() on null in vendor/magento/module-product-video/Model/Product/Attribute/Media/ExternalVideoEntryConverter.php:87" exception.
```

## Applicare la patch

Per applicare singole patch, utilizzare i collegamenti seguenti, a seconda del metodo di distribuzione utilizzato:

* Adobe Commerce su infrastruttura cloud: [Aggiornamenti e patch > Applica patch](https://experienceleague.adobe.com/docs/commerce-cloud-service/user-guide/develop/upgrade/apply-patches.html) nella guida Commerce su infrastruttura cloud.
