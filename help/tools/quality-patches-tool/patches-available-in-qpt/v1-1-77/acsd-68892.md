# ACSD-68892: comportamento di caching [!DNL Fastly] incoerente per le pagine memorizzabili in cache

La patch ACSD-68892 risolve il problema che impediva l&#39;archiviazione o la gestione corretta delle pagine memorizzabili nella cache dalla cache [!DNL Fastly], causando un comportamento di caching incoerente e prestazioni ridotte. Questa patch è disponibile quando è installato [[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.77. L’ID della patch è ACSD-68892. Questo problema è pianificato per la risoluzione in Adobe Commerce 2.4.9.

## Prodotti e versioni interessati

**La patch è stata creata per la versione di Adobe Commerce:**

* Adobe Commerce (tutti i metodi di implementazione) 2.4.8-p2

**Compatibile con le versioni di Adobe Commerce:**

* Adobe Commerce (tutti i metodi di implementazione) 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>La patch potrebbe diventare applicabile ad altre versioni con le nuove versioni di [!DNL Quality Patches Tool]. Per verificare se la patch è compatibile con la versione di Adobe Commerce in uso, aggiornare il pacchetto `magento/quality-patches` alla versione più recente e verificare la compatibilità nella pagina [[!DNL Quality Patches Tool]: Cerca patch](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=it). Utilizza l’ID patch come parola chiave di ricerca per individuare la patch.

## Problema

Le pagine memorizzabili in cache non vengono archiviate o servite correttamente dalla cache [!DNL Fastly], con conseguente comportamento di caching incoerente e prestazioni ridotte.

<u>Passaggi da riprodurre</u>:

1. Abilita la cache [!DNL Fastly].
1. Crea una nuova visualizzazione per siti Web/store/store.
1. Aggiungere il codice del sito Web secondario a `index.php`:

```
$_SERVER['MAGE_RUN_TYPE'] = 'website';
$_SERVER['MAGE_RUN_CODE'] = 'web2';
```

1. Controlla l’intestazione di controllo della cache utilizzando CURL:

```
curl -Is https://localhost/m247/pub/ | grep cache-control
```

<u>Risultati previsti</u>:

Le pagine memorizzabili in cache sono memorizzate nella cache in [!DNL Fastly].

<u>Risultati effettivi</u>:

Le pagine memorizzabili in cache non sono memorizzate nella cache in [!DNL Fastly], `max-age=0` restituito nell&#39;intestazione.

```
curl -Is https://localhost/m24develop/pub/ |grep cache-control
cache-control: max-age=0, must-revalidate, no-cache, no-store
```
